# Plan 02-02: Core Components

## Objective

홈 카탈로그 페이지에서 사용할 두 핵심 컴포넌트(JobCard, CategoryFilter)를 구현한다.
모두 Server Component로 작성하며, 기존 코드베이스의 CSS 변수 패턴(`--line`, `--muted`, `--panel`, `--foreground`)을 따른다. shadcn/ui는 설치되어 있지 않으므로 Tailwind v4 + CSS 변수 기반으로 구현한다.

또한 Phase 3에서 완성될 `/jobs/[slug]` 상세 페이지의 route stub을 추가하여 카드 클릭 시 404가 발생하지 않도록 한다.

    app/components/JobCard.tsx 파일을 신규 생성한다. (디렉토리 app/components/ 도 신규 생성.)

    컴포넌트 스펙:
    - import: next/link의 Link, @prisma/client의 Category
    - Props 타입: 인라인 타입 정의 (Prisma의 Job + category: Category 포함)
      필수 필드: id, slug, title, summary, skills(String), exampleCompanies(String),
                 salaryEntryMin(number|null), salaryEntryMax(number|null),
                 entryDifficulty(String), category: { name: string }

    내부 헬퍼 함수:
      function parseJsonArray(value: string): string[] {
        try { const p = JSON.parse(value); return Array.isArray(p) ? p : []; }
        catch { return []; }
      }

      function formatSalary(min: number | null, max: number | null): string | null {
        if (!min && !max) return null;
        if (min && max) return `${min.toLocaleString('ko-KR')}~${max.toLocaleString('ko-KR')}만원`;
        if (min) return `${min.toLocaleString('ko-KR')}만원~`;
        return `~${max!.toLocaleString('ko-KR')}만원`;
      }

    렌더링 구조 (Link로 감싸서 전체 카드 클릭 가능):
      <Link href={`/jobs/${job.slug}`}>
        <article className="...">
          상단: category.name (작은 녹색 텍스트 #246b57) + entryDifficulty 뱃지 (우측)
          중간: title (큰 폰트, bold) + summary (작은 텍스트, --muted)
          하단:
            - exampleCompanies: parseJsonArray(job.exampleCompanies).slice(0, 3).join(' · ')
              (값 없으면 렌더링 안 함)
            - skills: parseJsonArray(job.skills).slice(0, 3) 로 뱃지 렌더링
            - salary: formatSalary 결과가 있으면 "신입 " + 결과 텍스트로 표시

    스타일: 기존 page.tsx 카드 패턴 참고 (border border-[var(--line)], bg-[var(--panel)],
    rounded-lg, p-5). entryDifficulty에 따라 뱃지 색상 3종:
      "활발" → bg-[#edf6f1] text-[#246b57]
      "보통" → bg-[#fef8ec] text-[#9c6a00]
      "거의 없음" → bg-[#f2f2f2] text-[#555]

    전체 카드에 hover:shadow-md transition 추가.
    article에 min-h-56, flex flex-col justify-between 설정.
  </action>
