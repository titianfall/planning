# Plan 03-02: Job Detail Page — Components + Full Route

## Objective

`/jobs/[slug]` 상세 페이지 구현에 필요한 컴포넌트 4개를 신규 생성하고,
Phase 2에서 만든 route stub을 완전한 2컬럼 상세 페이지로 교체한다.

**컴포넌트 목록:**
- `MarkdownRenderer` — react-markdown + rehype-sanitize 기반 안전한 마크다운 렌더러
- `SkillTags` — 기술 태그 배지 목록
- `SalaryBlock` — 신입/경력 연봉 + 출처 링크
- `HiringDifficultyBadge` — 3단계 색상 배지 (커스텀 Tailwind, shadcn 없음)

**레이아웃 (D-01~D-03):**
- 데스크탑(1280px+): 헤더(full-width) + 본문(main, 좌 약 2/3) + 사이드바(aside, 우 약 1/3)
- 모바일(375px): 헤더 → 본문 → 사이드바 순서로 수직 스택
